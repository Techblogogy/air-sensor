# air-sensor

Sensor designed for collecting:
  - Levels of CO2 (in PPM)
  - Noise levels on street (in DB)
  - Dust density (in ??? )
  
 Required Hardware:
  - Arduino Uno board (or equivalent)
  - ESP8266 Wi-Fi Module (pins 5(TX), 6(RX))
  - DFRobots Analog Sound Sensor V2
  - Sharp Dust Sensor

### Made in Fabricator
